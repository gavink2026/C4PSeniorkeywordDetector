intended for Cyber4Project newcomers. Volunteers should install on computers for seniors using these steps.

Step 1: download files

Download the extension folder to your computer
Unzip it if it's in a ZIP file
Remember where you saved it

Step 2: Open Chrome Extensions

Open gc
chrome://extensions/ in the address bar and press Enter
turn on "Developer mode" - it's a switch in the top right corner

Step 3: load extension

Click the "Load unpacked" button
Find and select the extension folder you downloaded
Click "Select Folder"
Done! You'll see a shield icon in your browser toolbar


Installing for Someone Else (Seniors)
What You Need

Their computer with Chrome installed
The extension folder downloaded
5 minutes

Installation Steps

Download the extension to their Desktop
open Chrome on their computer
go to chrome://extensions/
turn on "Developer mode" (top right)
Click "Load unpacked"
select the extension folder from Desktop
pin the extension icon to their toolbar by clicking the puzzle piece and pin icon

Show Them How to Use It

Click the shield icon to open it
Copy and paste any suspicious message into the box
Click "Analyze Message"
Red means danger, green means safe
Read the explanation at the top

Important Notes

The extension will also check text they highlight on websites
They'll get a notification if something looks dangerous
All scanning happens on their computer, nothing is sent online
They can click the shield icon anytime to check a message
